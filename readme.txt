== Description ==
Adds a strip to the bottom of the screen for logged in admin users to allow quick access to popular parts on the Administration area in WordPress.<br/>
Dashboard<br/>
Logout<br/>
Write<br/>
Manage<br/>
Options<br/>
Akismet<br/>
Mint (Disabled using comments)<br/>
Number of posts and posts page<br/>
Number of comments and comments page<br/>
Number of categories and categories page<br/>
Number of tags and tags page<br/>
<a href="http://www.somefoolwitha.com">somefoolwitha.com</a>

== Installation ==
Installing should be a piece of cake and take fewer than five minutes.<br/>

Upload wp-adminstrip-2.php to your /wp-content/plugins/ directory.<br/>
Activate the plugin through the 'Plugins' menu in WordPress.<br/>
